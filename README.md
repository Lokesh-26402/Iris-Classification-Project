# Iris-Classification-Project
The Iris Classification Project is a machine learning project that classifies iris flowers into three different species — Setosa, Versicolor, and Virginica — based on their sepal and petal measurements. This project uses the famous Iris dataset and applies supervised machine learning algorithms to build an accurate flower classification model.
# Iris Classification Project

## Objective
Build a machine learning model to classify iris flower species.

## Algorithms Used
- KNN
- Logistic Regression
- Decision Tree

## Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1-score

## Best Model
KNN Classifier

## How to Run
1. Install dependencies
2. Run notebook
3. Load saved model
4. Perform predictions
